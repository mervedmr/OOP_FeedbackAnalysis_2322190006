# Customer Feedback and Sentiment Analysis Platform

This repository contains the Stage 1 architecture work for the Customer Feedback and Sentiment Analysis Platform.

### Stage 1 Goals
- Define core classes (Customer, Product, Feedback, ReviewAnalyzer)
- Demonstrate Aggregation (Product - many Feedback objects)
- Provide UML and design documents
- Prepare system structure for Stage 2

### Kendime Not:
- Stage 2’de CRUD işlemleri, sentiment analizi, kelime frekansı ve sıralama fonksiyonları eklenecek.
- Sınıfları sade bırak daha sonra genişletilecek.
